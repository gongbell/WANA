# Deploy on Docker
> Work in Progress...

> Our docker-compose file does not support **keosd** on v1.4.1. You can use **keosd** on your local machine.

```sh
# Deploy nodeos, keosd containers with docker-compose
docker-compose up -d

# Stops nodeos, keosd containers
docker-compose down
```
